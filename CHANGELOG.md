# Changelog

All notable changes to the Biz-Catalog theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-14

### Added
- Initial release of Biz-Catalog theme
- Generic, reusable template for business catalogs and portfolios
- Complete ACF integration with comprehensive field groups
- Custom post type "project" for portfolio items
- Custom taxonomy "service" for categorization
- Responsive design with mobile-first approach
- Swiper.js integration for project galleries
- Breadcrumb navigation system
- Contact form integration support
- SEO-friendly markup and meta tags

### ACF Field Groups
- **Site Settings**: Complete options page with company info, hero content, section titles, contact details
- **Service Fields**: Enhanced service taxonomy with image, description, icon, and color picker
- **Project Details**: Comprehensive project metadata including gallery, specifications, client info, featured flag

### Templates
- **front-page.php**: Dynamic homepage with ACF-powered sections
- **archive-project.php**: Project archive with filtering and sorting capabilities
- **single-project.php**: Detailed project view with gallery slider and metadata
- **taxonomy-service.php**: Service category archive pages
- Standard WordPress templates (404.php, search.php, etc.)

### Features
- Dynamic content management via ACF
- Featured projects system
- Service-based project filtering
- Responsive image galleries
- Contact information management
- Custom logo support
- Translation-ready structure
- WooCommerce compatibility
- Performance optimizations

### Documentation
- Comprehensive README with installation guide
- ACF field documentation
- Customization guidelines
- Contributing guidelines
- Issue templates for bug reports

### Technical Specifications
- WordPress 5.0+ compatibility
- PHP 7.4+ support
- GPL-2.0+ license
- Modern CSS with custom properties
- ES6+ JavaScript
- Accessibility-focused design

## Template History

### Previous Versions (Custom-Catalog)
- Legacy MR Furniture-specific branding and content
- Hardcoded contact information and content
- Limited customization options
- Basic project showcase functionality

### Migration Notes
- Theme renamed from "Custom-Catalog" to "Biz-Catalog"
- All hardcoded content replaced with ACF fields
- Text domain updated for generic reusability
- Enhanced field structure for better content management
