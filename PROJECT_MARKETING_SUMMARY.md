## üõ† Technical Architecture

### **Advanced Custom Fields (ACF) Integration**
**Three Comprehensive Field Groups:**

1. **Site Settings (22 fields)**
   - Company information, hero content, section titles
   - Contact details, social media, business facts
   - Footer copyright, form shortcodes

2. **Service Taxonomy Enhancement**
   - Hero banner images, short descriptions
   - Service icons, brand color pickers
   - Dynamic service categorization

3. **Project Details (8 fields)**
   - Image galleries with Swiper.js integration
   - Project specifications (repeater fields)
   - Client information, location, completion year
   - Budget tracking, featured project flags

### **Template Hierarchy**
- **front-page.php** - Dynamic homepage with ACF-powered sections
- **archive-project.php** - Filterable project catalog with sorting
- **single-project.php** - Detailed project showcase with galleries
- **taxonomy-service.php** - Service-based project filtering
- **Standard WordPress templates** - 404, search, archive pages

### **Modern Features**
- **Swiper.js Galleries** - Touch-enabled, responsive image sliders
- **Breadcrumb Navigation** - SEO-friendly page hierarchy
- **Contact Form Integration** - CF7 and Fluent Forms support
- **WooCommerce Compatible** - E-commerce compatible structure
- **Translation Ready** - Full internationalization support
